# Library 📕
```
1. 리눅스(우분투)
   서버 운영 환경으로 우분투를 사용하여 안정적이고 효율적인 서버 운영을 지원합니다.

2. 도커
   Docker를 사용해 여러 서버 인스턴스를 관리하고, 컨테이너화된 환경에서 배포하여 효율적인 서버 운영을 가능하게 합니다.

3. Nginx 
   로드 밸런싱을 구현하여 서버 트래픽을 분산시키고 성능을 최적화합니다.

4. C - 초기에 빠른 구현을 위해 사용했습니다.
   4-1. mongoose.h
      C의 웹 서버 및 웹 소켓 구현을 위한 경량 라이브러리인 mongoose.h를 사용해 효율적인 서버를 구현합니다.
   4-2. MySQL (MariaDB)  
      `mysql.h` 라이브러리를 사용하여 MariaDB 데이터베이스를 제어하고, 데이터를 효율적으로 처리합니다.
   4-3. JWT
      `jwt.h` 라이브러리를 사용하여 사용자 인증 및 토큰 관리를 구현합니다.
   4-4. cJSON 
      JSON 파싱을 위해 `cJSON.h` 라이브러리를 사용하여 효율적인 데이터 처리를 지원합니다.

5. Node.js - 브라우저가 아닌 환경에서 JavaScript 실행을 위한 런타임 환경, 더 확장성있고 생산성을 높이기 위해 사용했습니다.
   5-1. Express.js
      Node.js 기반에서 웹 서버를 쉽게 구현할 수 있도록 도와주는 프레임워크로 RESTful API 서버를 구축하고, HTTP 프로토콜에 따라 요청과 응답을 처리합니다.
   5-2. Socket.io  
      실시간 통신을 위한 웹 소켓 서버 구현에 `Socket.io` 라이브러리를 사용하여 클라이언트와 서버 간의 실시간 데이터 전송을 처리합니다.

위 기술 스택은 백엔드 서버와 웹 소켓 서버의 안정적인 운영을 위한 기반을 마련하고, 성능 최적화 및 효율적인 데이터 처리를 가능하게 합니다.

6. Markdown-UI
    Markdown 파일을 렌더링하고 사용자 인터페이스(UI)를 위한 마크다운 뷰어를 제공합니다.

```
